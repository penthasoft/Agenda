POS Promotional Coupon & Promotional Program

All IN ONE solution is provided to apply Coupons & Promotions codes on Sales, E-Commerce and Point of Sale.

This module is compatible with

  • Community
  • Enterprise
  • Odoo.sh

Key Features

    Provides option to apply coupon and promotion codes on Sales, E-Commerce and Point of Sale.
    Apply Promotional Coupon code & Promotional Program directly from Point of Sale Screen.
    Promotional Coupon / Promotional Program is directly applied on POS Invoice.
    POS promotional coupon is directly generated with receipt from POS machine, which could be applied on next order.
    On generating promotional coupon, email will be sent to customer to avail discount benefit.
    Warning message will appear if the coupon is used before or expired.
    Detailed Warnings are provided for Point of Sale.
    Statistical reports can be generated for Promotional coupon and promotion Program.

Notes

    This Application works with all Coupons & Promotions related scenarios and Logical conditions provided in Enterprise and Community editions.
    Coupons & Promotion codes generated for Sales, E-Commerce website can be easily used with Point of Sale.

Configuration - Coupon program

User can select options to apply coupon code on Sales order or Point of Sale or both.
Sale :- Coupons will only work with Sales and E-Commerce functionality.
Point of Sale :- Coupons will only work with Point of Sale.
Both :- Coupons will work for Sale, E-Commerce and Point of Sale.

Point of Sale User Configurations

To use promotional coupon and Promotional Program, select 'Apply Coupon & Promotion' option user wise in POS configuration.

'Coupon Code', 'Update Promotions' Option On POS Screen

'Coupon Code', 'Update Promotions' buttons will be visible on POS screen.

Applying Promotional Coupon code

From POS, after applying Coupon codes, resulted disconts will be automatically deducted from related Point of Sale order accordingly.

POS order receipt

Coupon code and discount applied will be printed on POS receipt.

Consumed Coupons

Used coupons will be automatically displayed as 'Consumed' in state from backend side so that they can not be used again.

Pos order

Point of Sale Order will be generated automatically with applied Coupon discount.

Configuration - Promotion Programs

User can select options to apply Promotion Program on Sales order or Point of Sale or both.
Sale :- Promotion Program will only work with Sales and E-Commerce functionality.
Point of Sale :- Promotion Program will only work with Point of Sale.
Both :- Promotion Program will work for Sale, E-Commerce and Point of Sale.

Applying Promotion Program code

On applying Promotional program code, discount will be applied directly on POS order.

POS order receipt

Promotion code and discount applied will be printed on POS receipt.

Pos order

Point of Sale Order will be reflected on backend side with related Promotion discount amount deducted.

Applying Promotion Code For Next Order

To configure promotion code for next order, select 'Apply on Next Order' Boolean.

Promotional program coupon with POS Receipt & automated Email

On selecting 'Apply on Next Order' Boolean, Promotional program coupon will be generated with POS receipt and an email will be sent to customer regarding benefits.

Pos order

Prevalent POS Order will not get affected because changes will be applied on the next order.

Warning Messages

When Promotional coupon is used after the expiry date, system will generate notification message.
If a coupon is already used in previous POS order, then system will display a notification warning message with details of POS Order reference,date & time for the coupon used.
If the coupon is used by another customer, the system will generate notification message with related customer name for the coupon to be used.

Statistical Reports

Statistical Reports can be easily generated with bifurcation of Coupons & Promotions.
