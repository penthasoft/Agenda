== Description ==
Agrega un campo de tipo RUT (Chileno) a Contact Form 7. Este plugin depende de Contact Form 7.

Al editar un formulario CF7, tendrá disponible un tipo de campo RUT. Este se incluye como cualquier otro campo en el editor del formulario. Al agregar el campo puede configurar diferentes opciones disponibles como son el “Auto formatear” o “Solo números”.

== Installation ==
Siga el procedimiento de instalación estándar para plugins de WordPress.
